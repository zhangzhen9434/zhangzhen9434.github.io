实例：1.0 通过用户输入两个数字，并计算两个数字之和：
  1. # -*- coding: UTF-8 -*-
  3. # Filename : SUM.py
  5. # 用户输入数字
  6. num1 = input('输入第一个数字：')
  7. num2 = input('输入第二个数字：')
  9. # 求和
  10. sum = float(num1) + float(num2)
  12. # 显示计算结果
  13. print('数字 {0} 和 {1} 相加结果为： {2}'.format(num1, num2, sum))

执行以上代码输出结果为：

  1. 输入第一个数字：1.5
  2. 输入第二个数字：2.5
  3. 数字 1.5 和 2.5 相加结果为： 4.0

其实，我们还可以将以上运算，合并为一行代码：

  1. # -*- coding: UTF-8 -*-
  3. # Filename : test.py
  4. # author by : www.runoob.com
  6. print('两数之和为 %.1f' %(float(input('输入第一个数字：'))+float(input('输入第二个数字：'))))

执行以上代码输出结果为：

  1. $ python test.py 
  2. 输入第一个数字：1.5
  3. 输入第二个数字：2.5
  4. 两数之和为 4.0
